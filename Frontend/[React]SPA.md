# SPA(Single Page Application)이란?
- html이 1개 뿐인 어플리케이션
- 전통적인 웹사이트는 페이지를 이동할 때마다, 서버에서 html, css, js(정적자원)을 내려준다면, SPA는 딱 한번만 정적자원을 받아옴.

## html을 하나만 주는 이유
- 중요한건 **사용성**
- 페이지를 이동할때마다 서버에서 html을 받으면, 상태유지 어려움

## 단점
- SPA는 딱 한번 정적자원을 내려받음
- 처음에 모든 컴포넌트를 받아옴.
- 사용자가 안 볼 페이지까지 모두 가져오기 때문에, 첫 로딩속도가 느려짐
